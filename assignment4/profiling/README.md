## Exercise 4.0 Time Profiling

Time profile programs that identify the slowest part (must be a function) of a program.

#### Contents :shipit:

* [manual_timing_.py](https://github.uio.no/IN3110/IN3110-nicoha/blob/master/assignment4/profiling/manual_timing_.py) - a more manual implementation of time profiling using `time.time`. The program generates a text file which states the measured times for all profiled functions, name the slowest/fastest part and compare the profiled functions. This program is generalized such that it can be used for the subsequent exercises in the assignment.

* [manual_report.txt](https://github.uio.no/IN3110/IN3110-nicoha/blob/master/assignment4/profiling/manual_report.txt) - text file with time profiling results generated by `manual_timing_.py`.

* [timeit_timing_.py](https://github.uio.no/IN3110/IN3110-nicoha/blob/master/assignment4/profiling/timeit_timing_.py) - implementation of time profiling using the `timeit` module. The program generates a text file which states the measured times for all profiled functions and name the slowest part.

* [timeit_report.txt](https://github.uio.no/IN3110/IN3110-nicoha/blob/master/assignment4/profiling/timeit_report.txt) - text file with time profiling results generated by `timeit_timing_.py`.

* [cProfile_timing_.py](https://github.uio.no/IN3110/IN3110-nicoha/blob/master/assignment4/profiling/cProfile_timing_.py) - implementation of time profiling using the `timeit` module. This program profiles a single function and generates a time report.

* [cProfile_report.txt](https://github.uio.no/IN3110/IN3110-nicoha/blob/master/assignment4/profiling/cProfile_report.txt) - text file with time profiling results generated by `cProfile_timing_.py`.

* [test_slow_rectangle.py](https://github.uio.no/IN3110/IN3110-nicoha/blob/master/assignment4/profiling/test_slow_rectangle.py) - example code to time profile with the different methods.


#### Usage :moyai:

`cd` into `profiling` directory.

**Run individual .py programs:**

    $ python <filename>.py

This will generate time profiling reports for functions specified by the particular program.
